# 🎌 MAHANIMO

> Seu universo de animes, mangás e manhwas.

---

# 📖 Sobre o projeto

O MAHANIMO é uma plataforma desenvolvida para reunir animes, mangás e manhwas em um único ambiente.

O projeto não hospeda conteúdos protegidos por direitos autorais. Seu objetivo é organizar informações, facilitar a descoberta de obras e direcionar o usuário para plataformas oficiais ou licenciadas.

O foco principal do projeto é oferecer uma interface moderna, organizada e agradável para consulta de conteúdos.
